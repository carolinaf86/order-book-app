# order-book-app

## Install
You need `node.js` and `yarn` installed globally  
`$ yarn # install deps`

## Run
`$ yarn start # run client`  
`$ node server.js # run server`  

Open [http://localhost:3000](http://localhost:3000)  
The page will reload as you make edits.
